## blog网址
https://jintaolu.github.io/

## 本地命令
创建新的文章
```
hexo new "文章标题"
```

清理缓存
```
hexo clean
```

生成静态网页
```
hexo g 或 hexo generate
```

启动服务
```
hexo s 或 hexo server
```

访问本地网页
```
http://localhost:4000/
```
